What Visuals Did We Use

Scatter Plots

Are used to plot data points on a horizontal and a vertical axis in the attempt to show how much one variable is affected by another.

Line Graphs

A typoe\ of a graphical visual that tracks changes over short and long periods of time.

Heat Map

A heat map is a data visualization technique that shows magnitude of a phenomenon as color in two dimensions.

Choropleth Map

A map that uses graded differences in shading of color on a map in order to indicate the average values of some property or quantity in those areas.

What Models Did We Use

Poly Regression

A form of regression analysis that uses the relationship between an independent variable and dependent variable and is modeled as a n nth degree polynomial curve

Support Vector Regression

A type of supervised learning algorithm that is used to predict discrete values and create a best fit line.

Gaussian

A classification machine learning algorithm that is used to classify data into different categories based on the probability distribution.

Lasso

LASSO (Least absolute shrinkage and selection operator) is a regression method that performs both variable selections and regularization to enhance the prediction accuracy and interpretability of the results.
